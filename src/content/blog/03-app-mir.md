Estudiar el MIR es duro. Junto con Miguel se nos ocurrió una app
que usara la API de Claude para hacernos la vida un poco más fácil.

## Qué hace la app

- Toma un documento base (la transcripción de las videoclases por tema)
- Lo cruza con el capítulo del manual correspondiente para reforzar lo relevante
- Crea flashcards si se las pides y las importa a Anki con un solo botón
- Tiene una pestaña para enviar documentos finales por Gmail desde la propia app, sin abrir el navegador
